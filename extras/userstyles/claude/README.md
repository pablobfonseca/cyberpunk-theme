## Variants

| Variant | Description |
|---------|-------------|
| **Storm** | Dark blues with full neon accents |
| **Night** | Deeper blacks with muted neon |
| **Neon** | Pure black with maximum saturation |

## Usage

1. Install the [Stylus](https://github.com/openstyles/stylus) browser extension
2. Click one of the install buttons above
3. Stylus will open and offer to install the theme

## Scope

Overrides Claude's native CSS custom properties (dark mode only). No class-name selectors are used, maximizing resilience to site updates.
